Information for how to run socket programming(Q5):-

1.open echo folder on one computer
2.terminal command: make

3.open client folder on another computer
4.terminal command: make
4.terminal command: java client <HOST_IP_ADDRESS> <PACKET_SIZE> <RC(=T)> <output_filename(with extension)>

(cumulative RTT and packet loss will print on client side terminal and datagram data(Sequence,time,RC,dummy value) would print on echo side terminal)
